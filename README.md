# Exercícios de Java

Este repositório contém exercícios básicos de Java desenvolvidos para praticar conceitos como entrada de dados, estruturas condicionais, operadores e switch-case.

## Lista de Exercícios

### 1. Olá Usuário
Peça o nome do usuário e imprima:  
`Olá, [nome], Seja bem vindo ao Java.`

### 2. Soma de dois números
Peça dois números inteiros e mostre a soma.

### 3. Área de um retângulo
Peça largura e altura e calcule:  
`área = largura * altura`.

### 4. Número positivo ou negativo
Peça um número e informe se ele é positivo, negativo ou zero.

### 5. Maior de idade
Peça a idade e informe se a pessoa é maior ou menor de idade.

### 6. Par ou ímpar
Peça um número inteiro e verifique se ele é par ou ímpar.

### 7. Calculadora simples
Peça dois números e uma operação (`+`, `-`, `*`, `/`).  
Use `switch` para calcular e mostrar o resultado.

### 8. Sistema de notas
Peça uma nota de 0 a 10 e mostre:  
- Reprovado (<5)  
- Recuperação (5 a 6.9)  
- Aprovado (≥7)

### 9. Dia da semana
Peça um número de 1 a 7 e mostre o dia correspondente usando `switch`.
